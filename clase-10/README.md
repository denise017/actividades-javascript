Clase 10 Repaso JavaScript

Ejercicio 1
Se declararon dos variables llamadas edad y peso y se les asignaron valores numericos para practicar el uso de variables

Ejercicio 2
Se pidieron al usuario su nombre edad y peso usando prompt y se mostro un mensaje personalizado con esa informacion en consola

Ejercicio 3
Se crearon dos constantes para edad minima y maxima y se valido la edad ingresada por el usuario usando parseInt y condicionales

Ejercicio 4
Se declararon variables con distintos tipos de datos boolean string number y null
Se realizaron operaciones concatenaciones y validaciones segun el tipo de dato

Ejercicio 5
Se declararon dos numeros y se realizaron operaciones matematicas como suma resta multiplicacion y division mostrando los resultados en consola

Ejercicio 6
Se usaron operadores de comparacion y logicos para verificar si dos numeros eran mayores que diez y se mostro el resultado

Ejercicio 7
Se intercambiaron los valores de dos variables y se mostraron los valores antes y despues del cambio

Ejercicio 8
Se pidio una temperatura en grados celsius y se convirtio a fahrenheit usando parseFloat y una formula matematica

Ejercicio 9
Se pidieron peso y altura al usuario y se calculo el indice de masa corporal usando una formula

Ejercicio 10
