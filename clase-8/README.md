# Clase 8 

## Actividad 1
Se trabajan situaciones de la vida cotidiana para identificar variables.

## Actividad 2

**parseFloat:** se usa para convertir lo que ingresa el usuario en un número.

## Actividad 3
Calculadora simple que pide dos números y una operación y muestra el resultado en consola.
